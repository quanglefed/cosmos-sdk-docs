# User Guides

Phần này được thiết kế cho các nhà phát triển đang sử dụng Cosmos SDK để xây dựng ứng dụng. Nó cung cấp các hướng dẫn và tài liệu tham khảo thiết yếu để sử dụng hiệu quả các tính năng của SDK.

*   [Setting up keys](./run-node/00-keyring.md) - Tìm hiểu cách thiết lập quản lý khóa bảo mật bằng tính năng keyring của Cosmos SDK. Hướng dẫn này cung cấp phương pháp xử lý khóa mật mã được tối ưu hóa, rất quan trọng để bảo mật ứng dụng của bạn.
*   [Running a node](./run-node/01-run-node.md) - Hướng dẫn này cung cấp các bước chi tiết để triển khai và quản lý node trong mạng Cosmos. Nó đảm bảo hoạt động mượt mà và đáng tin cậy của ứng dụng blockchain bằng cách bao gồm tất cả các bước thiết lập và bảo trì cần thiết.
*   [CLI](./run-node/02-interact-node.md) - Khám phá cách điều hướng và tương tác với Cosmos SDK bằng Command Line Interface (CLI). Phần này bao gồm các thao tác dựa trên lệnh hiệu quả và mạnh mẽ có thể giúp bạn quản lý ứng dụng một cách hiệu quả.
